
--Creazione Database--

Database ricavato da un file txt preesistente ("ricette.txt"), contenente 28.200 ricette, tramite l'utilizzo di un programma Python ("conv.py") da noi creato.
Quest'ultimo trasforma il file txt in un file csv ("ricette.csv") con appositi delimitatori da noi scelti e lo rende pronto quindi ad essere importato come database in Postgres grazie ad uno script sql ("copy.sql").
